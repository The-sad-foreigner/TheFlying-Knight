# TheFlying-Knight
Рыцарь не летает,  
он прыгает по платформам.

## Пролог:  
это мир Сельрии в котором  
бесчинствовали демоны, но силой нации  
большинство демонов было убито и  
остались лишь летающие демоны которые  
собрались в башне лорда летающих  
демонов до которой не сможет добраться  
ни армия, ни магия. В итоге люди  
призвали героя чтобы он искоренил  
демонов с этих земель.  

### Жанр:  
приключение, подземелье.  

## Геймплей:  
Вы прыгаете по платформам и убиваете  
демонов прокачивая уровень. Вы можете  
собирать снаряжение и торговаться с  
жителями на земле.  
игнор
